Using Atom

Download Atom

Go to atom.io to download. If you're using a Mac, put it in your Applications folder.

Shortcuts

Keyboard shortcuts are absolutely worth learning. If you take some time to learn them, it will make your life much easier.

  • ⌘ + A Select all
  • ⌘ + Z Undo
  • ⌘ + C Copy
  • ⌘ + X Cut
  • ⌘ + V Paste
  • ⌘ + S Save
  • ⌘ + W Close tab
  • ⌘ + , Settings
  • ⌘ + \ Toggle tree view
  • tab Indent code
  • shift + tab Outdent code
  • ⌘ + shift + D Duplicate line
  • ⌘ + shift + P Command Palette - You can perform most commands from here

Use File > Add Project Folder to add a project folder to work on. You can also drag and drop the project folder onto your Atom dock icon.

Use Edit > Lines > Auto Indent to make sure that your code is properly indented.

File Management

Do:

  • Make a folder to keep all your work in for this class. Call it j363 or something similar.
  • Use the 'column view' in the Finder (Mac Users)
  • Use the 'list view' in Windows Explorer (PC Users)
  • Keep backups of your files. You are expected to use a service like Google Drive or Box to maintain your work.

Don't:

  • Keep all of your stuff on your desktop
  • Keep only one copy of your work. You should have your files backed up in at least one place

Naming Conventions

Use short, descriptive filenames. Avoid capital letters and spaces.
My Project.html is a poor file name.
my-project.html is a better file name.

Recommended Settings

Under the Editor tab: make sure Show Indent Guide is checked. Set Tab Length to 4.
