Project 1

Project 1 will test students' knowledge of basic HTML, MarkDown, and CSS.
This project has two parts:

(40 pts.) Part 1 - Basic page structure with HTML and MarkDown
(60 pts.) Part 2 - Creating a multi-page site using HTML and CSS

Part 1 - Structure

You will use a Markdown and HTML for this part. Remember that Markup is another word for HTML (Hypertext Markup Language)

Recall that Markdown is a tool With Markdown, using this syntax:

# This is an h1
## This is an h2

* This
* is
* a
* list

Will give you:

<h1>This is an h1</h1>
<h2>This is an h2</h2>

<ul>
    <li>This</li>
    <li>is</li>
    <li>a</li>
    <li>list</li>
</ul>
