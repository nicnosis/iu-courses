Week 7

  • We will be using Emmet to improve workflow in SublimeText. Emmet gives us shortcuts to generate HTML quickly.
    • To install Emmet:
    • Bring up the package control in SublimeText. Use CMD+SHIFT+P and look for 'install package'
    • Look for 'Emmet' and install it.
    • To use Emmet:
    • Type your desired element and hit TAB
    • For example, typing 'p' and pressing tab will generate a paragraph tag.
    • Typing '.row' and pressing tab will generate a div with the 'row' class.
    • For more information, see the Emmet Documentation
    • SitePoint also has a great tutorial.

  • We will also be using the View in Browser package to accelerate workflow.
    • To install View in Browser:
    • Bring up the package control in SublimeText. Use CMD+SHIFT+P and look for 'install package'
    • Look for 'Emmet' and install it.
    • To use View in Browser:
    • Press CTRL+OPTION+C to view your HTML file in Chrome
