J360 - Web Design

This page will host the course content, links, and other materials for MSCH-J360, Web Design.

Week 1

  • Read - The syllabus
  • Read - W3 Schools HTML Tutorial - up to and including section 'HTML Styles'

Week 2

  • Install - SublimeText.
  • Read - W3 Schools HTML Tutorial - up to and including section 'HTML Lists'
  • Complete - MarkDown Tutorial
  • Practice - Using Dillinger.io and Commonmark to generate HTML
  • Create an account - on CodePen
  • Read - W3 Schools: HTML Classes
  • Visit - CSS Zen Garden and choose a style. Download the HTML and CSS example files and try altering the CSS to create visual changes.

Week 3

  • Install - Sublime Package Control
  • Install - MarkdownEditing.
    See section "Package Control" for installation instructions
  • Also install - Markdown Preview for Sublime
  • (this is also done with the package manager)
  • Download - this image for our in-class activity
  • Read - Block and inline elements on W3Schools.

Week 4

  • We will begin our work on Project 1.
  • Read - CSS Box Model

Week 5

  • Week 5 will be used to review prior concepts, work on Project 1, and introduce Bootstrap
  • Review this checklist to ensure that you are caught up.
  • Begin reading - W3Schools: Bootstrap Tutorial
  • Read - Bootstrap - Grid Basics
  • Download - Bootstrap starter template
  • Download - Bootstrap Gaia Theme template

Week 6

  • By this point, you should feel comfortable with:
    • The CSS Box Model
    • Using classes to style elements
    • Rows and columns in Bootstrap
    • File-name conventions
  • Week 6 features some CodePen projects used for in-class activities.
    • 6.1 - CSS Box Model
    • 6.2 - CSS Display Property
    • 6.3 - Bootstrap Basics
